# MediQueue Backend

This is the backend server for MediQueue.  
Built with Express.js and MongoDB.

## Features

- REST API
- Better Auth
- Google Login
- Tutor Management
- Booking System
- MongoDB Database

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Better Auth

## Run Locally

```bash
npm install
npm start
```

## Environment Variables

```env
PORT=5000
MONGODB_URI=your_mongodb_uri
CLIENT_URL=http://localhost:3000
BETTER_AUTH_SECRET=your_secret
BETTER_AUTH_URL=http://localhost:5000
```
